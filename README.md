# SEB - Scanning Extensions Bot
Advanced Discord forensic tool for indexing server attachments and video embeds.

## Key Features
Granular Filtering: Scan by category *(Video, Audio, Code, etc.)*, specific channels, or timeframes.
Smart Exclusions: Skip multiple channels by tagging them directly in the command.
Reporting: Generates detailed *.txt* logs with jump links and file types, sent to channel or DM.|

## Usage
Run `/scans` with your desired parameters **(requires Administrator permissions)**.
